StartupLens AI-Powered Startup Ecosystem Platform Product Requirements Document (PRD) 1. Product Overview StartupLens is an AI-powered startup ecosystem platform designed for founders, investors, and developers. It provides a central hub to discover startups, evaluate ideas using AI, match founders with investors, and manage community events. 2. Problem Statement Startup ecosystems suffer from fragmented tools and manual processes. Founders need validation, investors need efficient filtering, and the community needs a unified connection point. 3. Target Users - Founder: Wants AI feedback on their pitch. - Investor: Wants to discover and filter relevant startups. - Developer: Wants to find co-founders or join early-stage startups. 4. Feature Specifications - Dashboard: Live stats, quick actions, recent activity. - Startup Profiles (CRUD): List/detail view with industry/stage filters. - Investor Discovery (CRUD): Search and filter investors by focus area/stage. - AI Startup Evaluator: Input pitch -> Gemini AI -> Structured JSON evaluation (Scores, Strengths, Improvements, Verdict). - AI Founder-Investor Matching: Rank investors by fit score for a specific startup. - Events: Networking, pitch, and workshop listings. 5. Technical Stack Next.js, Firebase, Shadcn UI, Tailwind CSS, Google Gemini 1.5 Flash.
